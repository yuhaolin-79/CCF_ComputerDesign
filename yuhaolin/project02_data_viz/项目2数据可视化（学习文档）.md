## 1.2 绘制简单的折线图
```python
import matplotlib.pyplot as plt

squares = [1, 4, 9, 16, 25]

fig, ax = plt.subplots()
ax.plot(squares)

plt.show()
```

![[squares_01.png]]

>**`subplot()`函数可以在一个图形中绘制一个或多个绘图(plot)，变量`fig`表示一系列绘图构成的整个图形，变量`ax`表示图形中的绘图。**

### 1.2.1 修改标签文字和线条粗细

### 1.2.4 使用`scatter()`绘制散点图并设置样式
```python
import matplotlib.pyplot as plt

x_values = list(range(1, 1001))
y_values = [x**2 for x in x_values]

plt.style.use('seaborn-v0_8')
fig, ax = plt.subplots()
ax.scatter(x_values, y_values, s=10)

# Set chart title and label axes.
ax.set_title("Square Numbers", fontsize=24)
ax.set_xlabel("Value", fontsize=14)
ax.set_ylabel("Square of Value", fontsize=14)

# Set size of tick labels.
ax.tick_params(labelsize=14)

# Set the range for each axis.
ax.axis([0, 1100, 0, 1100000])

plt.show()
```

>**`s`用来设置点的大小**

### 1.2.5 定制刻度标记
- 数据较大时，Matplotlib 默认使用科学计数法，当然可以使用常规计数法

```python
ax.ticklabel_format(sytle='plain')
```

### 1.2.6 定制颜色

```python
ax.scatter(x_values, y_values, color='red', s=10)
```

- 也可以使用RGB色
- `color=(0, 0.8, 0)`传递一个元组，三个0-1的浮点数，表示红色、绿色、蓝色。
- 越接近0，颜色越深，越接近1，颜色越浅

### 1.2.7 使用颜色映射
```python
ax.scatter(x_values, y_values, c=y_values, cmap=plt.cm.Blues, s=10)
```

### 1.2.8 自动保存绘图

- 只需将`plt.show()`替换为`plt.savefig('name.png', bbox_inches='tight')
- 如果要保留多余的空白区域，第二个实参可以省略
- 当然也可以使用`path`，将其保存到指定位置

```python
plt.savefig('./pictures/SquareNums08.png')
```

## 1.3 随机游走
```python
import matplotlib.pyplot as plt

from random_walk import RandomWalk
# Create a random walk instance
rw = RandomWalk(5000)
rw.fill_walk()

# plot all points
plt.style.use('classic')
fig, ax = plt.subplots()
ax.scatter(rw.x_values, rw.y_values, s=15)
ax.set_title("Random Walk", fontsize=24)
ax.set_aspect('equal')

plt.show()
```

### 1.3.2 设置随机游走图的样式
#### 1.绘点着色
```python
import matplotlib.pyplot as plt

from random_walk import RandomWalk
while True:
    # Create a random walk instance
    rw = RandomWalk(5000)
    rw.fill_walk()

    # plot all points
    plt.style.use('classic')
    fig, ax = plt.subplots()
    point_numbers = range(rw.num_points)
    ax.scatter(rw.x_values, rw.y_values, c=point_numbers, cmap=plt.cm.Blues,
            edgecolors='none', s=15)
    ax.set_title("Random Walk", fontsize=24)
    ax.set_aspect('equal')

    plt.show()
    keep_running = input("Make another walk? (y/n): ")
    if keep_running.lower() == 'n':
        break
```

#### 2.重新绘制起点和终点
```python
# Emphasize the first and last points
    ax.scatter(0, 0, c='green', edgecolors='none', s=100)
    ax.scatter(rw.x_values[-1], rw.y_values[-1], c='red',
            edgecolors='none', s=100)
```

#### 3.隐藏坐标轴
```python
# Remove the axes
    ax.get_xaxis().set_visible(False)
    ax.get_yaxis().set_visible(False)
```

#### 5.调整尺寸以适应屏幕大小

