# 项目2：数据可视化（蟒蛇书学习实践）

## 一、项目说明
本文件夹是《Python编程：从入门到实践》（蟒蛇书）**数据可视化部分的学习实践仓库**，记录数据可视化基础技能的学习过程与代码成果，后续会持续补充更多可视化案例（如交互式图表、数据仪表盘等）。


## 二、已完成学习内容
当前已掌握Matplotlib核心基础技能，包含：
1. **图表类型**：散点图、折线图的绘制与样式配置；
2. **样式管理**：Matplotlib内置样式（如`seaborn-v0_8`）的使用；
3. **图表元素配置**：标题、坐标轴标签、刻度大小的自定义；
4. **图片保存**：指定路径（`pathlib`/`os.path`）保存高清图表（去留白+300DPI）；
5. **数据可视化流程**：从数据生成→图表绘制→样式优化→结果保存的完整流程。
