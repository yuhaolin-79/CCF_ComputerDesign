# 🐍 Python 七天速成计划
本项目记录了鱼浩琳同学的 Python 学习历程，目标是通过 7 天时间快速掌握 Python 核心语法与开发技巧。

---

## 📅 第一阶段：基础语法与规范 (Day 1)
*重点掌握变量类型、列表操作及编写规范。*
- **变量与字符串**：f-字符串格式化、大小写转换、删除空白/前缀后缀。
- **数值运算**：整数/浮点数运算差异、下划线增强大数字可读性。
- **列表基础**：增删改查（`append`/`insert`/`pop`/`remove`）、排序。
- **列表进阶**：`for`循环遍历、`range()`、切片、列表推导式。
- **元组**：不可变序列的定义与使用。
- **PEP 8 规范**：4空格缩进、行长度限制、命名准则（变量小写下划线/常量全大写）。

## 📅 第二阶段：逻辑控制与交互 (Day 2)
*重点掌握流程控制、复杂数据结构及用户交互。*
- **if 语句**：条件测试、`if-elif-else`、多条件`and/or`、`in/not in`判断。
- **字典**：键值对增删改查、`get()`安全访问、遍历（`items()`/`keys()`/`values()`）。
- **数据嵌套**：字典列表、字典嵌套列表/字典。
- **用户输入**：`input()`获取信息、`int()`类型转换。
- **While 循环**：标志位/`break`/`continue`、动态处理列表/填充字典。

## 📅 第三阶段：函数定义与使用 (Day 3)
*重点掌握函数的定义、参数传递及模块化调用。*
- **函数基础**：`def`定义、文档字符串、函数调用。
- **参数传递**：位置实参、关键字实参、默认值。
- **返回值**：`return`返回任意类型、可选形参。
- **列表与函数**：传递列表/列表副本、批量处理。
- **可变参数**：`*参数`（元组）、`**参数`（字典）处理任意数量参数。
- **模块化导入**：导入模块/指定函数、别名使用。

## 📅 第四阶段：类、文件操作与异常处理 (Day 4)
*重点掌握类、文件读写及异常捕获处理。*
- **类**：
  - 类的定义（大驼峰命名）、实例化创建对象；
  - 核心__init__初始化方法，创建对象时自动执行，实现实例属性的批量初始化，self为必传参数，代表当前实例本身；
  - 定义实例方法（含self参数），实现对象的专属功能，通过对象.方法()调用；
  - 实例属性的定义与访问，通过self.属性名绑定、对象.属性名获取 / 修改。
- **文件操作**：
  - `open()`（`r`/`w`/`a`/`r+`）、`with`语句自动关闭文件；
  - `read()`/`readline()`/`write()`读写、`os`模块处理路径；
  - `json.dump()`/`json.load()`实现数据持久化。
- **异常处理**：
  - `try-except`捕获指定异常（`FileNotFoundError`等）；
  - `else`/`finally`分支、`raise`主动抛异常。

## 📅 第五阶段：文件异常与单元测试 (Day 5)
*重点掌握异常处理进阶、JSON数据存储及pytest单元测试完整流程。*
- **异常处理进阶**：`pass`语句实现静默失败、异常场景的无提示处理。
- **JSON数据存储**：`json.dumps()`序列化数据、`json.loads()`反序列化数据、跨程序数据共享。
- **用户数据持久化**：`Path.exists()`判断文件存在性、结合JSON实现用户数据保存与读取。
- **代码重构**：将功能代码拆分为单一职责函数、提升代码可读性与可维护性。
- **pytest环境搭建**：`pip`升级方法、`pytest`的用户级安装、`py -m pytest`无环境变量运行方式。
- **函数单元测试**：`test_`开头的文件/函数命名规范、`assert`断言的基础使用、测试通过/失败的结果解析、多场景测试用例编写。
- **类的单元测试**：自定义类的功能测试、多断言验证类的方法逻辑、覆盖单/多数据场景。
- **pytest核心特性**：常用断言语法、`@pytest.fixture`夹具实现测试对象复用、搭建通用测试环境。
