# 🐍 Python 七天速成计划

本项目记录了鱼浩琳同学的 Python 学习历程，目标是通过 7 天时间快速掌握 Python 核心语法与开发技巧。

---

## 📅 第一阶段：基础语法与规范 (Day 1)
*重点掌握变量类型、列表操作及编写规范。*

* **变量与字符串**：掌握 f-字符串格式化、常用字符串处理方法（大小写转换、删除空白、前缀后缀处理）。
* **数值运算**：理解整数与浮点数的运算差异，学习使用下划线增强大数字的可读性。
* **列表基础**：熟练进行列表元素的增删改查（`append`, `insert`, `pop`, `remove`）及永久/临时排序。
* **列表操作进阶**：使用 `for` 循环遍历、`range()` 生成数值列表、切片（slice）以及列表推导式。
* **元组**：理解不可变序列的定义与应用场景。
* **PEP 8 规范**：严格遵守 4 个空格缩进、行长度限制、函数间空行以及规范的命名准则（变量小写加下划线，常量全大写）。

## 📅 第二阶段：逻辑控制与交互 (Day 2)
*重点掌握流程控制、复杂数据结构及用户交互。*

* **if 语句**：掌握各种条件测试（数值比较、多个条件 `and`/`or`、检查元素是否在列表中）以及 `if-elif-else` 结构。
* **字典 (Dictionary)**：学习键值对的操作，包括动态添加、修改、删除及使用 `get()` 安全访问值。
* **数据嵌套**：掌握字典列表、在字典中存储列表以及字典套字典的复杂结构处理。
* **用户输入**：使用 `input()` 获取信息并利用 `int()` 进行数值类型转换。
* **While 循环**：利用标志位或 `break`/`continue` 控制循环，并学习如何使用 `while` 动态处理列表和填充字典。

## 📅 第三阶段：函数定义与使用 (Day 3)
*重点掌握函数的定义、参数传递及模块化调用。*

* **函数基础**：使用 `def` 定义函数，添加文档字符串，掌握函数调用方式。
* **参数传递**：掌握位置实参、关键字实参、默认值的使用技巧。
* **返回值**：使用 `return` 返回任意类型数据，支持可选形参实现灵活返回。
* **列表与函数**：向函数传递列表（或副本），实现列表的批量处理。
* **可变参数**：掌握 `*参数`（元组）、`**参数`（字典）处理任意数量参数。
* **模块化导入**：学习导入模块、指定函数及别名使用的规范方式。
